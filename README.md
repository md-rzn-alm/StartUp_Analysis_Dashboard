# 🚀 Startup Funding Analysis Dashboard

An interactive **Startup Funding Analysis Dashboard** built using **Python, Streamlit, Pandas, Plotly, and SQL concepts** to analyze investment trends in the Indian startup ecosystem. The dashboard enables users to explore startup funding, investor activities, industry performance, and city-wise investment patterns through interactive visualizations and insightful analytics.

## 🌐 Live Demo

👉 https://startup-insights-dashboard.streamlit.app/

---

# 📌 Project Overview

This project transforms raw startup funding data into an interactive business intelligence dashboard. Users can explore funding trends, identify top investors and startups, analyze sector-wise investments, and gain valuable insights into the Indian startup ecosystem.

The dashboard is designed for:

- Data Analysts
- Business Analysts
- Investors
- Startup Enthusiasts
- Students learning Data Analytics

# 📂 Dataset

The project uses an Indian Startup Funding dataset containing information such as:

- Startup Name
- Industry
- Investors
- Funding Amount
- Funding Round
- City
- Date
- Investment Type

After preprocessing, the dataset contains:

- **2,451 Startups**
- **2,400+ Investors**
- **₹3.49 Lakh Crore Total Funding**

---

# 📈 Key Insights

Some interesting business insights generated from the dashboard:

- 💰 Total funding exceeds **₹3.49 Lakh Crore**
- 🏙️ **Bengaluru** is the leading startup investment hub.
- 🛒 **E-commerce** received the highest funding among all sectors.
- 🚀 **Rapido Bike Taxi** emerged as the highest-funded startup.
- 📅 **2017** recorded the highest annual funding.
- 📆 **August 2019** witnessed the highest monthly funding.
- 📊 Average funding per startup is approximately **₹142 Crore**.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/startup-funding-dashboard.git
```

Navigate to the project folder

```bash
cd startup-funding-dashboard
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
